# Getting started
- `npm install`
- `bower install`
- `npm start` (`npm run dev` for nodemon)

# Run unit tests
- `npm run test:unit`

# Run E2E tests
- Make sure you have selenium running
    Note: You can download chrome webdriver from http://chromedriver.storage.googleapis.com/index.html?path=2.9/
- Open a new terminal
- `npm run test:e2e`
